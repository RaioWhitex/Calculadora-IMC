# ⚖️ Calculadora de IMC

Uma calculadora de Índice de Massa Corporal (IMC) que recebe peso e altura, calcula o IMC e exibe a classificação correspondente, com interface limpa e responsiva.

Ideal para acompanhamento de saúde e prática de cálculos e exibição de resultados em tempo real.

## ✨ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 🎨 Características do Design

- Visual leve e moderno
- Paleta de cores com destaque em rosa
- Campos de entrada com cantos arredondados
- Botão de ação em destaque
- Tabela de classificação organizada
- Layout centralizado e responsivo

## 🚀 Objetivo do Projeto

Praticar a construção de interfaces interativas, explorando captura de valores, cálculo de IMC e exibição dinâmica de resultados com JavaScript, além da estilização com CSS.

## 📚 Funcionalidades

- Campos para inserção de peso e altura
- Cálculo automático do IMC
- Exibição do resultado com a classificação
- Tabela de referência com faixas de IMC
- Botão para refazer o cálculo
- Interface leve e responsiva

## 💼 Possíveis Aplicações

- Calculadora de IMC online
- Ferramenta de acompanhamento de saúde
- Componente para sites de fitness e bem-estar
- Widget para landing pages
- Recurso educativo sobre saúde
